# Meus Comandos Favoritos
Aqui estão os comandos que mais utilizei na aula de Terminal:

- 'cd': para navegar entre as pastas
- 'cls': para apagar todo texto visível da tela
- 'mkdir': para criar pastas
- 'dir': para listar as pastas/arquivos criados
- 'cd..': para voltar um nível no diretório
